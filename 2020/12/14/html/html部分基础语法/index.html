html部分基础语法

发表于 2020-12-14 分类于 html

html基础语法总结

常用标签

排版标签
h

标题标签

p

段落

hr

水平线

br

换行

div

盒子 , 没有语义

span

盒子 , 没有语义

文本格式化标签
b/strong

文字以粗体方式显示(XHTML推荐使用strong)

i/em

文字以斜体方式显示(XHTML推荐使用em)

s/del

文字以加删除线方式显示(XHTML推荐使用del)

u/ins

文字以加下划线方式显示(XHTML推荐使用ins)

img

<img src="图像URL" />

a

<a href="跳转目标" target="目标窗口的弹出方式">文本或图像</a>

注释

<!-- 注释语句 --> 快捷键是： ctrl + / 或者 ctrl +shift + /

锚点

base 标签

<base target="_blank" />

**总结： **

  1. base 可以设置整体链接的打开状态
  2. base 写到 之间
  3. 把所有的连接 都默认添加 target=”_blank”

pre

预格式化文本标签。
在 pre 标签 元素中的文本通常会保留空格和换行符。而文本也会呈现为等宽字体。

部分符号

table

定义一个表格

tr

定义表格中的行

td

定义表格中的单元格，必须嵌套在 tr 标签中

cellspacing

设置单元格与单元格边框之间的空白间距
平时开发cellspacing 为 0

cellpadding

设置单元格内容与单元格边框之间的空白间距
平时开发cellpadding 为 0

th

表头单元格标签,一般表头单元格位于表格的第一行或第一列，并且文本加粗居中
用th替代相应的td即可。

caption

定义表格标题，通常这个标题会被居中且显示于表格之上。
caption 标签必须紧随 table 标签之后。

合并单元格
  • 跨行合并：rowspan=”合并单元格的个数”
  • 跨列合并：colspan=”合并单元格的个数”

列表

只讲下自定义列表

表单

input 控件

<input type="属性值" value="你好">

label标签

label标签主要目的是为了提高用户体验。

  • 第一种用法就是用label直接包括input表单, 适合单个表单选择

<label> 用户名： <input type="radio" name="usename" value="请输入用户名"> </label>

  • 第二种用法 for 属性规定 label 与哪个表单元素绑定
textarea控件

通过textarea控件可以轻松地创建多行文本输入框

select下拉列表

form表单域

在HTML中，form标签被用于定义表单域，以实现用户信息的收集和传递，form中的所有内容都会被提交给服务器。
